Methodology

    
The data is generated by a Ruby script that scans all reviewer prompts and their accompanying JSON records in the _reviewers directory. It counts languages, categories, repositories, and comment activity, then stores the aggregated numbers in assets/data/trends.json. A GitHub Actions workflow runs this script on every push so the charts always reflect the latest data.
